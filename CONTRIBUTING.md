# Contribuer à Afriland Workspace

Merci de l'intérêt que vous portez à **Afriland Workspace** ! Nous sommes ravis de recevoir de l'aide pour construire cet espace de travail.

## Comment aider ?

Vous pouvez contribuer de plusieurs manières :

- En signalant des bugs.
- En proposant de nouvelles fonctionnalités.
- En améliorant la documentation.
- En répondant aux issues étiquetées `help wanted` ou `good first issue`.

## Processus de contribution

1. **Forkez** le projet.
2. **Créez une branche** pour votre modification (`git checkout -b feature/nom-ma-fonctionnalite`).
3. **Faites vos changements** et testez-les.
4. **Commitez** vos modifications avec un message clair.
5. **Poussez** vers votre branche (`git push origin feature/nom-ma-fonctionnalite`).
6. Ouvrez une **Pull Request**.

## Installation locale

Pour faire tourner le projet sur votre machine :

```bash
# Cloner le projet
git clone [https://github.com/TON_NOM_UTILISATEUR/afriland-workspace.git](https://github.com/TON_NOM_UTILISATEUR/afriland-workspace.git)

# Installer les dépendances avec Bun
bun install

# Lancer le serveur de développement
bun dev
```

## Stack Technique

- **Framework :** Next.js (App Router)
- **Runtime :** Bun
- **Auth & Database :** Supabase
- **ORM :** Prisma

## Règles de conduite

Soyez respectueux et bienveillants dans vos échanges. Nous sommes ici pour apprendre et construire ensemble !
